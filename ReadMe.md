## Projects Overview

### 1. Car Booking Application - [react-app]
- **Purpose**: Learn basic React concepts through a practical booking system
- **Key Features**:
  - Car booking form
  - Dynamic car card display
  - Garage page navigation
- **Learning Outcomes**:
  - Component creation and organization
  - State management using useState
  - Conditional rendering
  - Basic routing concepts

### 2. Profile Card Application [react-2]
- **Purpose**: Understanding props and component styling
- **Key Features**:
  - Display user information
  - Dynamic hobby list
  - Responsive design
- **Learning Outcomes**:
  - Props and PropTypes
  - CSS styling in React
  - Event handling
  - Component reusability
